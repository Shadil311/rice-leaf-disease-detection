# Rice Leaf Disease Detection using Deep Learning

## Project Overview
This project uses deep learning to classify rice leaf diseases from images. 
The model detects three diseases:
- Bacterial Leaf Blight
- Brown Spot
- Leaf Smut

## Dataset
120 rice leaf images organized into three classes.

## Model
Transfer learning using:
- MobileNetV2
- EfficientNetB0
- InceptionV3

Best model accuracy: **82.6%**

## Technologies Used
Python, TensorFlow, Keras, Streamlit, NumPy, Matplotlib

## Application
A Streamlit web app allows users to upload a rice leaf image and get disease prediction.
